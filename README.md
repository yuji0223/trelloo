Trelloo & Share
====
https://trelloo-app.herokuapp.com/

## 概要
Trelloライクなカンバン型タスク管理ツール + ユーザー全員共有ページで「やりたいこと・趣味・紹介したいものや動画」などを自由に共有できるアプリケーションです

## 制作背景
- 前職でタスク管理にスマホのメモアプリ、手帳、時にはPCに付箋を貼って管理する事があり、それらを一つのツールでまとめたいと思いました。
- タスク管理ツールについて調べてみると Trello という便利なアプリケーションが存在することを知り、＋αの機能を持たせたものを作ることに決めました。
- スマートフォンに「いつかやりたいこと」を、昔からメモしていました。ある日、友人も同じように管理している事を知り、色々な人のやりたい事を一覧で見ることができたら面白いなと思いました。
- 私はいろいろなエンタメに興味があり、twitterでフォロワーに紹介するほどではないですが、誰かと共有したいと思っていました。私と同じように趣味を気軽に紹介したい人もいると思い、本アプリケーションを開発しました。

## 機能
- ユーザー登録・ログイン・ログアウト
- テストユーザーログイン
- リスト作成・編集・削除
- リスト内にカード作成・編集・削除
- カードのリスト間移動
- ドラッグ&ドロップによるカードの並び替え
- Shareページ
  - いいね機能
  - コメント機能
